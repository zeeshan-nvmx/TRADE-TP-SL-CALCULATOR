<objective>
Add per-field taker/maker fee mode selectors so the user can independently set whether each leg (entry, TP exit, SL exit, trailing SL exit) uses the taker or maker rate.

Purpose: Traders who exit with limit orders (maker) pay less than market order exits. This lets the calculator reflect any combination of entry/exit order types accurately.
Output: Four fee mode state vars in useCalculator; four Taker/Maker toggle buttons in TradeSetup below the exchange selector.
</objective>
